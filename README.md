# Evo Fintech

## Live Demo
 https://evo.mykhalskyio.xyz/swagger/index.html



## Requirements
* Git
* Docker
* golang-migrate 


## How to run with MakeFile

Clone repository

    $ git clone https://github.com/mykhalskyio/image-api.git

Build and Run

    $ make build

Run migrate up

    $ make migration-up



## Endpoints
* POST   - /api/upload 
* GET    - /api/download/json
* GET    - /api/download/csv
* GET    - /swagger/index.html
